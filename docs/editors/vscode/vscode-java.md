# VS Code setup for Java

[← back](index.md)

Set up VS Code for building and running Java apps.

## 1 Install JDK

[Instruction for Linux](../../operating-systems/linux/linux-java.md) - Install JDK for Linux  
[Instruction for Windows](../../operating-systems/windows/windows-java.md) - Install JDK for Windows

## 2 Install VS Code extension

Install the extension `Extension Pack for Java` (official from Microsoft)

It includes:
* Java Language Support
* Debugger for Java
* Maven/Gradle Tools

## 3 Run java program

Use `F5` key to execute java files.
