# General

## 1. Set up color theme

Set up a window color scheme:  
`Settings > Window Color Scheme > Breeze Dark`

Set up an editor color theme:  
`Settings > Editor Color Theme > RS Dark` (my custom theme)

To import custom theme:  
* Go to `Settings > Configure Kate > Color themes > Theme Editor` and press `Import`
* choose your theme file (`RS Dark.theme`)

Download my custom theme: [RS Dark](assets/RS%20dark.theme)


## 2. Set up tabulations

Make this if you want to use Tabs instead of Spaces.  
Navigate to:  
`Settings > Konfigure Kate > Editing > Indentation`

In the `Indent using` section choose `Tabulators`

## 3. Editor

To enable "scroll past end" feature:  
Navigate to:  
`Settings > Konfigure Kate > Editing > Text Navigation`

Set the `Allow scrolling past the end of the document` checkbox.
