# Ollama installation

[← back](index.md)

## Installation

### Windows

Download installer:
```
https://ollama.com/download/OllamaSetup.exe
```

or install from PowerShell:

```powershell
irm https://ollama.com/install.ps1 | iex
```

### Linux

Install:

```bash
curl -fsSL https://ollama.com/install.sh | sh
```

## Check version:

```sh
ollama -v
```
