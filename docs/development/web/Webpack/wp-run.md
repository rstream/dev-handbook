## Build

Run build in default mode (specified in `webpack.config.js`):
```sh
webpack
```

Run build in dev mode:
```sh
webpack --mode development
```

Run build in production mode:
```sh
webpack --mode production
```

## Watch

Run watch on the local server:
```sh
webpack --watch
```
