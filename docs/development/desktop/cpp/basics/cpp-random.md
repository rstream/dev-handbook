# Random generation in old C/C++

[← back](../index.md)

Before modern `<random>`, C and old C++ code usually used `rand()` and `srand()`.

This approach is simple, but limited.
Use it when you are learning old code, working with legacy examples, or need very basic random values.

For modern C++ code, prefer [`<random>`](../std/cpp-std-random.md).

## 1. Idea

`rand()` returns a pseudo-random integer.

```cpp
int value = rand();
```

Pseudo-random means that values look random, but they are generated by an algorithm.

By default, many implementations start from the same seed.
This means the program can produce the same sequence every run.

## 2. Include headers

`rand()` and `srand()` are declared in `<cstdlib>`.

```cpp
#include <cstdlib>
```

For `time()` you need `<ctime>`.

```cpp
#include <ctime>
```

For input and output:

```cpp
#include <iostream>
```

## 3. Random integer

Basic example:

```cpp
#include <cstdlib>
#include <iostream>

int main()
{
    int value = rand();

    std::cout << value << '\n';

    return 0;
}
```

`rand()` returns an integer from `0` to `RAND_MAX`.

```cpp
std::cout << RAND_MAX << '\n';
```

`RAND_MAX` depends on compiler and platform.

## 4. Seed random generator

Use `srand()` once at the start of the program.

```cpp
#include <cstdlib>
#include <ctime>
#include <iostream>

int main()
{
    srand(time(nullptr));

    int value = rand();

    std::cout << value << '\n';

    return 0;
}
```

`time(nullptr)` returns current time in seconds.
It is commonly used as a simple changing seed.

Important:

```cpp
srand(time(nullptr));   // do this once
```

Do not call `srand()` before every `rand()`.

## 5. Range from 0 to N

Use the remainder operator `%`.

```cpp
int value = rand() % 10;
```

This gives values from `0` to `9`.

For a dice:

```cpp
int dice = rand() % 6 + 1;
```

This gives values from `1` to `6`.

## 6. Range from min to max

Formula:

```cpp
int value = min + rand() % (max - min + 1);
```

Example:

```cpp
int min = 10;
int max = 20;

int value = min + rand() % (max - min + 1);
```

This gives values from `10` to `20`.
Both limits are included.

## 7. Random character

To choose one character, generate a random index.

```cpp
const char letters[] = "abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ";

int len = sizeof(letters) - 1;   // do not count final '\0'
int index = rand() % len;

char c = letters[index];
```

`letters` is a C string, so it has a final `'\0'`.
That is why `sizeof(letters) - 1` is used.

## 8. Random C string

A practical legacy task is creating a random text suffix, temporary name, or test identifier.

This example creates a random string from Latin letters, one character at a time.

```cpp
#include <cstdlib>
#include <ctime>
#include <iostream>

void randomString(char result[], int length)
{
    const char letters[] =
        "abcdefghijklmnopqrstuvwxyz"
        "ABCDEFGHIJKLMNOPQRSTUVWXYZ";

    int lettersLen = sizeof(letters) - 1;

    for (int i = 0; i < length; i++) {
        int index = rand() % lettersLen;
        result[i] = letters[index];
    }

    result[length] = '\0';
}

int main()
{
    srand(time(nullptr));

    char text[13];
    randomString(text, 12);

    std::cout << text << '\n';

    return 0;
}
```

Example output:

```text
PuKqZadxQmLe
```

`char text[13]` has space for:

* 12 generated characters
* final `'\0'`

Without final `'\0'`, the result is not a valid C string.

## 9. Random password-like token

Use a larger character set when numbers are also allowed.

```cpp
#include <cstdlib>
#include <ctime>
#include <iostream>

void randomToken(char result[], int length)
{
    const char chars[] =
        "abcdefghijklmnopqrstuvwxyz"
        "ABCDEFGHIJKLMNOPQRSTUVWXYZ"
        "0123456789";

    int charsLen = sizeof(chars) - 1;

    for (int i = 0; i < length; i++) {
        result[i] = chars[rand() % charsLen];
    }

    result[length] = '\0';
}

int main()
{
    srand(time(nullptr));

    char token[17];
    randomToken(token, 16);

    std::cout << token << '\n';

    return 0;
}
```

This is good for demos, test data, and non-critical identifiers.

Do not use this for real passwords, sessions, reset links, or API keys.

## 10. Common mistakes

Wrong: seeding inside a loop.

```cpp
for (int i = 0; i < 10; i++) {
    srand(time(nullptr));
    std::cout << rand() << '\n';
}
```

`time(nullptr)` changes only once per second.
This can restart the generator with the same seed many times.

Correct: seed once.

```cpp
srand(time(nullptr));

for (int i = 0; i < 10; i++) {
    std::cout << rand() << '\n';
}
```

Wrong: forgetting space for `'\0'`.

```cpp
char text[12];
randomString(text, 12);   // overflow: needs 13 chars
```

Correct:

```cpp
char text[13];
randomString(text, 12);
```

## 11. Limitations

Legacy `rand()`:

* has weak random quality
* has small range on some platforms
* has awkward range conversion with `%`
* is not suitable for security-sensitive values
* uses global state

For modern C++ code, use [`<random>`](../std/cpp-std-random.md) with `std::mt19937` and distributions.
